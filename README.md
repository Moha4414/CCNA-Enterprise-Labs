# CCNA Enterprise Labs

A collection of Cisco CCNA networking labs built in Cisco Packet Tracer.

This repository documents my journey from networking fundamentals to enterprise-level network design through hands-on labs and real-world scenarios.

## Objectives

* Learn networking through practical implementation
* Master Cisco CCNA topics
* Develop troubleshooting skills
* Build a professional networking portfolio
* Document network designs using industry-standard practices

## Technologies

* Cisco Packet Tracer

* VLANs

* Network Security
